# Deployment Guide

This site is designed to be deployed on Vercel and will replace zachmcnair.com.

## Quick Deploy

1. Connect your GitHub repository to Vercel
2. Set the following environment variables:
   - `LASTFM_API_KEY` (optional, for music scrobbler)
   - `LASTFM_USERNAME` (optional, for music scrobbler)

## Domain Setup

1. Point zachmcnair.com to Vercel
2. Update DNS settings to point to Vercel's servers
3. Enable HTTPS automatically

## Features

- ✅ Modern, responsive design
- ✅ Dark mode support
- ✅ SEO optimized
- ✅ Last.fm integration (optional)
- ✅ Blog functionality
- ✅ Case studies
- ✅ RSS feeds
- ✅ Open Graph images
- ✅ Sitemap generation

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Content Updates

- Blog posts: `app/content/blog/`
- Case studies: `app/content/case-studies/`
- Main page content: `app/page.tsx`

## Customization

- Fonts: Faktum family (included)
- Colors: Tailwind CSS classes
- Layout: Responsive grid system
- Analytics: Vercel Analytics (included)
